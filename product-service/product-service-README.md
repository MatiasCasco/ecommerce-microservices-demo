# 📦 Product Service

## 🧠 Responsibility
Gestión de productos, precios y stock.

---

## 🔄 Flow

- Consulta de productos
- Validación de stock
- Actualización de inventario

---

## 📡 Endpoints

### GET /products
Listar productos

### GET /products/{id}
Detalle producto

### POST /products
Crear producto
